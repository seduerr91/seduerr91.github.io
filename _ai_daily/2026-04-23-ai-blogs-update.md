OpenAI has launched ChatGPT workspace agents for Business, Enterprise, and Edu users, emphasizing automation of repeatable workflows across integrated tools. These agents streamline team processes—such as preparing meeting briefs by pulling data from apps like Google Calendar and SharePoint, and leveraging web search—while supporting scheduling, memory for context, and easy sharing within workspaces. Users can customize agents via a conversational builder with granular skills and permissions, ultimately enhancing efficiency, collaboration, and productivity across teams.

### [chatgpt-agents-sales-meeting-prep.md](https://github.com/openai/openai-cookbook/blob/564a630c0bdfa56b18a87f0f63e17cd18af9f202/articles/chatgpt-agents-sales-meeting-prep.md)
**Source:** openai/openai-cookbook

The blog post discusses the introduction of ChatGPT workspace agents, designed for Business, Enterprise, and Edu users. These agents can automate repeatable workflows across multiple tools, enhancing efficiency for teams. The article outlines a step-by-step guide to creating a workspace agent tailored for a sales team, which automates the preparation of meeting briefs by gathering relevant customer data from integrated apps like Google Calendar and SharePoint, as well as providing web search functionalities. Key features include scheduling the agent to run daily, memory capabilities for contextual continuity, and the ability to share agents within workspaces. Users can customize agents using a conversational builder, ensuring high-quality output through the use of skills and permissions. Overall, the agents aim to streamline processes, facilitate collaboration, and improve productivity within teams.
